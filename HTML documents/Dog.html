Creating An Implicit Link

1.Find a picture of a dog online and download it. Name the file as you choose, and ensure to place it in the “HTML documents” folder on your desktop.
2. Open a new Notepad++ file and create a new webpage (new HTML file) that includes a picture of a dog (using the tag).
3. Save this file as Dog.html within your “HTML documents” folder on your desktop.
4. Open your Assignment.html file and using the anchor element and href attribute, link your Assignment.html webpage to the Dog.html webpage.
